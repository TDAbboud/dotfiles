# Dotfiles

## Contents

Configurations for:
+ vim
+ emacs
+ sublime text
+ tmux
+ git
+ osx
+ Homebrew files (brew.sh)

## Install

1. `git clone https://github.com/tdabboud/dotfiles.git ~/.dotfiles`
2. `cd ~/.dotfiles`
3. `./install.sh`

## Custom Settings

Place all custom shell/exports/env settings in a file named `~/.custom`
